## Package - Paquete

* Paquete raíz de todo el proyecto se crea a partir del nombre completo del paquete con la url de la empresa cliente o desarrolladora en orden inverso

### Ejemplo

* Estructura básica

```java
dominio.empresa.ejemplo
```

* Estructura definida

```java
com.rvsweb.app.tutorial
```
